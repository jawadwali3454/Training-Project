# 📊 EduPredict - Student Performance Prediction

## 🔍 Project Overview
EduPredict is a Machine Learning project that predicts students' final grades (G3) using academic and social factors.

## 🧠 Machine Learning Models Used
- Linear Regression
- Random Forest Regressor

## 📂 Dataset
UCI Student Performance Dataset

## 🎯 Features Used
- studytime
- failures
- absences
- G1 (First period grade)
- G2 (Second period grade)

## 📈 Results
- Linear Regression and Random Forest were compared
- Random Forest performed better with:
  - Lower MAE (~1.05)
  - Higher R² Score (~0.87)

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 🚀 How to Run
1. Open notebook in Google Colab
2. Upload dataset
3. Run all cells

## 📊 Visualization
- Correlation Heatmap
- Actual vs Predicted Graph
- Model Comparison Charts

## 📌 Conclusion
Random Forest Regressor performed better and is selected as the final model.
